# 4.11 DNS

### Задание 1. 

Может ли компьютер работать без `DNS`? Будет ли работать сеть если в ней нет `DNS-сервера`? 

*Приведите ответ в свободной форме.*

### Ответ:
- Может ли компьютер работать без `DNS`? 

Да, компьютер может работать без DNS.
- Будет ли работать сеть если в ней нет `DNS-сервера`? 

Да, сеть будет работать если в ней нет DNS-сервера.

Но, придется использовать IP-адреса для доступа к различным ресурсам сети.

*Приведите ответ в свободной форме.*

### Ответ:
`/etc/hosts`

### Задание 4. 

Для чего служит ресурсная DNS запись типа `MX`?

*Приведите ответ в свободной форме.*

### Ответ:
`MX`-запись отвечает за сервер, через который будет работать электронная почта.
